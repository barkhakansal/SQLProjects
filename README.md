📊 SQL Projects Portfolio

This repository contains a collection of SQL-based data analysis projects focused on extracting insights from 
real-world datasets. The projects demonstrate skills in data cleaning, transformation, and exploratory data analysis
using SQL.

🛠️ Tools & Technologies
SQL (MySQL)
Python (Pandas) (for some projects)
Data Cleaning & Preprocessing
Data Analysis & Aggregation

📁 Projects Included
1. COVID-19 Data Analysis

Analyzed COVID-19 data to identify trends in cases, deaths, and recovery rates. Used SQL queries to explore patterns 
across regions and time.

2. Data Cleaning Project 

Performed data cleaning operations including handling missing values, removing duplicates, and transforming columns to
improve data quality.

3. Diabetes Data Analysis

Explored healthcare data to identify patterns related to diabetes. Used SQL queries to analyze patient metrics and risk 
factors.

4. Glycemic Variability Analysis

Analyzed blood glucose variability data to understand fluctuations and potential health implications.

5. Movie Dataset Analysis (Maven Movies)

Performed analysis on movie data to uncover trends in revenue, genres, and ratings using SQL queries.

6. World Bank Loan Data Analysis

Conducted end-to-end data analysis using Python and SQL, including data cleaning, transformation, and 
querying to uncover trends in global loan distribution and repayment.

🔍 Key Skills Demonstrated
Writing complex SQL queries (JOINs, GROUP BY, aggregations)
Data cleaning and preprocessing
Identifying trends and patterns in data
Working with real-world datasets

🚀 Summary
These projects showcase foundational to intermediate-level data analytics skills, 
with a focus on solving real-world problems using SQL and data-driven approaches.
